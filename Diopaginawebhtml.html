O que são tag?

voltar

As tags HTML são como comandos que ajudam a organizar e mostrar informações em uma página em um navegador são elementos usados ​​para marcar e formatar os conteúdos: textual ou visual em documentos HTML. Elas servem para dizer ao navegador o que é um título, um parágrafo, uma imagem, ou um link, além de também adicionar tais informações. São escritas com um formato específico, e cada uma tem um propósito diferente e são cercadas pelos símbolos de < ou >. É fascinante como tudo começa com algo tão simples!

Títulos de h1 até h6:

voltar

Essas tags criam títulos na página, com h1 sendo o maior e mais importante, até h6, que é menor e menos relevante. São ótimas para organizar o texto em níveis diferentes de hierarquia.

Parágrafos, a tag p:

voltar

A tag p formata o texto em parágrafos. Quando usada, ela coloca o texto em blocos separados, com espaçamento automático, o que deixa tudo mais organizado.

Tag mark, marcando informações IMPORTANTES!

voltar

A tag mark realça partes do texto, colocando um fundo colorido, como se fosse um marcador fluorescente. Deve ser útil para chamar atenção para informações importantes.

Tag small:

voltar

A small deixa o texto menor, ideal para algo como notas de rodapé ou detalhes que não precisam de tanto destaque.

Itálico, usando a tag i:

voltar

O i, usado como tag, coloca o texto em itálico. Parece boa para destacar palavras ou expressões sem exagerar no visual. Bem prática!

Sublinhado, usando a tag u:

voltar

A tag u é usada para sublinhar o texto. Pode ser útil para destacar algo importante de maneira visual, mas sem mexer no tamanho ou peso do texto.

Deixando o texto em negrito usando strong:

voltar

A tag strong deixa o texto em negrito e parece dar um significado especial, como se estivesse dizendo: "Ei, isso aqui é importante!".

Lista ordenada, ol:

voltar

Tag ol faz listas numeradas. Ela facilita criar sequências de passos ou listas ordenadas de forma automática.

Lista não ordenadas, ul:

voltar

A tag ul é bem parecida com a ol, mas sua função é fazer listas com marcadores, tipo pontos. Boa para informações que não precisam de uma ordem específica.

Itens da lista, li:

voltar

Tag li. Essa define cada item dentro das listas. Ela funciona tanto em listas ordenadas quanto nas não ordenadas.

Links a tag a:

voltar

Tag a serve para criar links que podem levar para outras páginas, arquivos ou seções. É uma das tags mais importantes para navegar na web.

Linha divisória, a tag hr:

voltar

A tag hr insere uma linha horizontal na página, o que ajuda a dividir o conteúdo visualmente. Simples e super útil para organização.

Escrevendo um pouco abaixo da linha, a tag sub:

voltar

A tag sub coloca texto em índice inferior. Acho que seria muito útil para coisas como fórmulas químicas ou explicações adicionais.

Escrevendo um pouco a cima da linha, a tag sub:

voltar

A tag sup é o oposto da sub. Ela coloca o texto em índice superior. Deve ser perfeita para potências,como "22".

Blockquote:

voltar

E por fim a tag blockquote, que é usada para destacar citações. Além de separar visualmente, também dá um tom mais formal ou destacado às palavras citadas.
